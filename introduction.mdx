## What is Pipedream?

Pipedream is a hosted platform where developers connect APIs and build event-driven automations. With over **2,800 integrated apps** and **10,000+ pre-built components**, you can automate workflows across any service — or write custom Node.js, Python, Go, or Bash code when you need full control.

## How it works

Pipedream workflows are linear sequences of steps that run when an event occurs.

<Steps>
  <Step title="Choose a trigger">
    Select a trigger to start your workflow. Options include HTTP/webhooks, schedules (cron), app events (new GitHub issue, new Slack message), or email.
  </Step>
  <Step title="Add steps">
    Add pre-built actions to interact with any of Pipedream's 2,800+ integrated apps. Or write custom Node.js, Python, Go, or Bash code.
  </Step>
  <Step title="Deploy and run">
    Deploy your workflow and Pipedream runs it every time the trigger fires. View execution logs, inspect event data, and iterate quickly.
  </Step>
</Steps>
