## Welcome to Pipedream

Pipedream is an integration platform for developers. Connect APIs, automate workflows, and build event-driven applications — all without managing infrastructure.

## What can you build?

<CardGroup cols={2}>
  <Card title="Workflow Automations" icon="diagram-project" href="/workflows/overview">
    Automate any sequence of steps triggered by HTTP requests, schedules, or app events — no servers required.
  </Card>
  <Card title="Event-Driven Apps" icon="bolt" href="/sources/overview">
    React to real-time events from GitHub, Slack, Stripe, Airtable, RSS, and 2,800+ other apps.
  </Card>
  <Card title="Custom Integrations" icon="code" href="/concepts/custom-code">
    Write Node.js, Python, Go, or Bash when you need full control. Import any package from npm, PyPI, or the Go module registry.
  </Card>
  <Card title="Embedded Integrations" icon="puzzle-piece" href="/connect/overview">
    Let your users connect their own accounts to 2,800+ APIs through your product using Pipedream Connect.
  </Card>
</CardGroup>
